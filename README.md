# 老健向けAIシフト作成システム

入所者100名規模の介護老人保健施設を想定した、営業デモ用のWebアプリです。看護師10名・介護士28名のダミー職員を自動生成し、選択した月の勤務表、配置チェック、職員別集計、不足・警告をまとめて作成します。

## 主な機能

- 対象月を選んで1か月分のシフトを自動作成
- 月間シフト表と、選択日の出勤者一覧をワンタッチで切り替え
- 日別出勤者を勤務区分別のカードで表示し、看護師・介護士を分けて確認
- 必要人数を最優先し、全勤務区分を不足0で自動配置
- 7〜10日周期の循環編成で、夜勤翌日の「明け」、明け翌日の「休み」を自動設定
- 連続勤務、月休日、夜勤回数、土日勤務の公平性を確認
- 将来、人数や条件を変更して不足が生じた場合は赤色で表示
- 5シート構成のExcelファイルをダウンロード

## 動作に必要なもの

- Python 3.10以上（3.11または3.12推奨）
- インターネット接続（初回のライブラリ導入時のみ）

## 実行方法

### 1. このフォルダを開く

ターミナル（WindowsではPowerShell）を開き、このREADMEと `app.py` があるフォルダへ移動します。

### 2. 必要なライブラリをインストールする

```bash
pip install -r requirements.txt
```

環境によって `pip` が見つからない場合は、次をお試しください。

```bash
python -m pip install -r requirements.txt
```

### 3. アプリを起動する

```bash
streamlit run app.py
```

`streamlit` が見つからない場合は、次でも起動できます。

```bash
python -m streamlit run app.py
```

ブラウザが自動的に開きます。開かない場合は、ターミナルに表示された `Local URL`（通常は `http://localhost:8501`）をブラウザで開いてください。

### 4. アプリを終了する

ターミナルで `Ctrl + C` を押します。

## 使い方

1. 「対象月」で作成する月を選びます。
2. 「AIシフトを生成」を押します。
3. 表示形式で「月間シフト表」または「日別出勤者一覧」を選びます。
4. 日別出勤者一覧では、確認したい日付を選び、勤務区分別のカードを確認します。
5. 下部のタブで日別配置、職員別集計、不足・警告を確認します。
6. 「Excelファイルをダウンロード」を押して保存します。

## Excelのシート構成

1. 月間シフト表
2. 日別配置チェック表
3. 職員別集計表
4. 不足・警告一覧
5. 日別出勤者一覧

## ご注意

必要人数の充足を最優先するため、現在の職員数では月9日の「休み」を満たさない職員が発生します。該当者は不足ではなく「注意」として一覧に表示されます。

このアプリは営業デモ用です。実際の勤務表として運用する際は、職員ごとの雇用条件、希望休、資格・スキル、配置加算、法令、施設独自ルールなどを追加し、責任者が最終確認してください。
